# 🧭 User Journey — DevVerse

---

## 🟢 New User Flow

1. User visits website
2. Clicks "Sign Up"
3. Options:
   - Email + Password
   - Google Login (Clerk)

4. Profile Setup:
   - Username
   - Bio
   - Social links

---

## 🔵 Returning User Flow

1. User opens app
2. JWT checked in localStorage
3. If expired → refresh token used
4. Auto-login

---

## 👥 Collaboration Flow

1. User creates project
2. Invites others via link
3. Multiple users join
4. Real-time editing begins
5. AI monitors conflicts

---

## 👤 Profile Interaction

1. Visit user profile
2. View projects
3. Follow user
4. Check social links

---

## ⚡ Code Execution Flow

1. User writes code
2. Clicks "Run"
3. Backend spins container
4. Output returned

---

## 🤖 AI Interaction

1. User writes code
2. AI detects issue
3. Suggestion popup appears
