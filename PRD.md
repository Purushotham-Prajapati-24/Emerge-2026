# 🚀 DEVVERSE — Cloud-Based Collaborative IDE

## 📌 Overview
DevVerse is a cloud-based collaborative IDE designed for 100+ simultaneous users with real-time editing, AI assistance, and social developer profiles.

---

## 🎯 Problem Statement
Existing collaborative IDEs:
- Do not scale well beyond small teams
- Lack intelligent conflict handling
- Have poor developer identity/social layer
- No ownership tracking or contribution visibility

---

## 💡 Solution
DevVerse provides:
- Real-time collaborative coding (100+ users)
- AI-powered code orchestration
- Developer social profiles (like GitHub + IDE)
- JWT + Clerk hybrid authentication
- Cloud execution environment

---

## 🧑‍💻 Target Users
- Hackathon teams
- Open-source contributors
- College coding groups
- Remote dev teams

---

## 🧩 Core Features

### 1. Real-Time Collaborative Editor
- Multiple users editing simultaneously
- Cursor tracking
- Conflict resolution (AI-assisted)

### 2. Authentication System
- Email/password login (JWT)
- OAuth (Google via Clerk)
- Secure session handling

### 3. Developer Profiles
- Username, bio, skills
- Projects list
- Followers / following
- Social links (GitHub, LinkedIn)

### 4. AI Code Assistant
- Suggest fixes
- Detect conflicts
- Code optimization hints

### 5. Cloud Execution
- Run code in sandbox
- Multi-language support

- Successful collaboration demo
